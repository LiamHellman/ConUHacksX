# Factify Chrome Extension

AI-powered text analysis for detecting bias, logical fallacies, and more - right in your browser.

## Features

- **Text Selection Analysis**: Select any text on a webpage and instantly analyze it
- **Bias Detection**: Identify political, emotional, and cognitive biases
- **Fallacy Detection**: Spot logical fallacies and weak arguments
- **Ethical Concerns**: Highlight potentially misleading or harmful content
- **Tone Analysis**: Understand the emotional tone of the text
- **Credibility Score**: Get an overall assessment of text reliability

## Installation

### From the Factify Website
1. Visit [factify.tech](https://factify.tech)
2. Click "Get Chrome Extension"
3. Follow the installation prompts

### Manual Installation (Developer Mode)
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked"
5. Select the `extension` folder

## Usage

1. **Select text** on any webpage
2. Click the **"Analyze with Factify"** button that appears
3. View the results in the popup panel
4. Or right-click selected text and choose **"Analyze with Factify"**

You can also click the extension icon in your toolbar to analyze text you've copied.

## Privacy

- Text is sent to Factify's secure API for analysis
- No data is stored permanently
- No browsing history is collected
